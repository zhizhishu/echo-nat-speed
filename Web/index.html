NAT 类型说明

NAT1

完全圆锥型 (Full Cone)

连接性最佳，内网主机可以被外部任意主机主动连接。P2P 下载、游戏主机联机完全畅通无阻。

NAT2

受限圆锥型 (Restricted Cone)

连接性良好，内网主机必须先向特定的外部 IP 发送过数据，该外部主机才能向内网主机发送数据。

NAT3

端口受限型 (Port Restricted)

连接性一般，限制精确到端口级别。外部主机必须匹配此前内网主机通信过的 IP 和端口才能回传数据。

NAT4

对称型 (Symmetric NAT)

连接性最差，每次与不同的外部目标通信时，路由器都会分配一个新的端口。P2P 打洞极难成功，通常需要 TURN 中继。
